# BondQuest

## A Gamified Web Application for Ionic and Covalent Bonding: Design and Quantitative Learning Gain Analysis

### Features
- Student Login
- Teacher/Admin Login
- Pre-test & Post-test
- XP, Coins & Badges
- Leaderboard
- Progress Tracking
- Learning Gain Analysis
- Interactive Chemistry Lessons
- Teacher Dashboard

- JavaScript
- Node.js
- Express.js
- MySQL
- JWT
- bcrypt
- Chart.js
